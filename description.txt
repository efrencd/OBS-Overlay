OBS Overlay Controller
======================

Esta aplicación permite controlar un overlay en OBS mediante un servidor WebSocket sencillo, dos páginas HTML y un conjunto de sonidos.

Descripción
-----------
- `server.js`: servidor `ws` que rebroadcasts mensajes JSON recibidos.
- `obs.html`: overlay para OBS que muestra un lower-third y reproduce sonidos según los comandos.
- `ext.html`: panel de control que se conecta al servidor, envía presets y dispara efectos de sonido desde botones rosas.
- `sounds/`: carpeta con efectos grabados que el overlay reproduce cuando recibe `sound` especificado.

Instrucciones
-------------
1. Instala dependencias:
   ```
   npm install
   ```
2. Arranca el servidor WebSocket:
   ```
   npm start
   ```
3. Abre `ext.html` en el navegador (puedes usar otro equipo en la LAN) y ajusta la URL del servidor si es necesario antes de reconectar.
4. Usa los presets o los botones de sonido para enviar instrucciones al overlay.
5. Abre `obs.html` como fuente `Browser` dentro de OBS (ajusta tamaño a 1920x1080) para ver el lower-third y escuchar los sonidos.
